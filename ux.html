Game Design

Asteroids

The goal of this project was to mimic the old-school arcade version of the game. Additionally, save/load features were implemented to become familiar with Unity's serialization process.

This game was made to be played with Xbox for Windows controllers. Turn, accelerate and decelerate with the left stick and fire with the right trigger. Escape functionality has not been added to this game so to quit, you must force quit.

Runner

For this assignment, our class was asked to create a simple platformer.
The emphasis for this project was placed on creating a helpful user interface and proper physics mechanics

The game will run for you. To jump, press the space bar. P allows you to access the menu.

Catapult

This game was an introduction to physics simulation.

Use the mouse to drag the ball and launch it towards the scene. Press escape to reset the scene. This game must be exited using force quit.
